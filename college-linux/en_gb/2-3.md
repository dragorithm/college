### Creating Partitions

```bash
sudo apt install gdisk
sudo sgdisk -n 0:0:0 /dev/sda
```

### Creating a Filesystem with mkfs

```bash
sudo mkfs.vfat -F 32 /dev/sda1
```

### Checking a Filesystem with fsck

```bash
sudo fsck.vfat /dev/sda1
```

### Deleting Disk Partitions

```bash
sudo sgdisk -d 1 /dev/sda
sudo sgdisk --label-clear /dev/sda
```

## Other Disk Management Tools

Using other disk management tools (e.g. `dd`, `df`, `du`, `mount`) for disk optimization and configuration.

### dd

```bash
sudo dd if=/dev/zero of=/dev/sda bs=512 count=2048
dd if=/dev/zero of=disk.img bs=1M count=100
```

- **Command 1**: Writes zeros to /dev/sda 2048 times, 512 bytes per write
- **Command 2**: Creates a virtual disk image file

### du

```bash
du /root
du -sh /home
sudo du -sh /* | sort -h | tail -n 10
```

Parameter notes:

- `-s`: Summary
- `-h`: Human-readable output (GB/MB)

### mount

```bash
mkfs.ext4 disk.img
sudo mkdir /mnt/vdisk
sudo mount disk.img /mnt/vdisk
```

### df

```bash
df -h
df -h /mnt/vdisk
```

## RAID

```bash
su
apt install -y zfsutils-linux zfs-dkms linux-headers-amd64
```

### Kernel

*Note*: When rebooting, select a kernel image that supports ZFS.

```bash
root@debian:~# modprobe zfs
modprobe: FATAL: Module zfs not found in directory /lib/modules/6.12.74+deb13+1-amd64
```

The `kernel 6.12.74+deb13+1-amd64` shown above is the one you should **not** select at boot.

### raidz1

```bash
zpool create -f -o ashift=12 tank raidz1 /dev/sda /dev/sdb /dev/sdc /dev/sdd
```

### zvol

```bash
zfs create -V 5G tank/xfs
apt install -y xfsprogs
mkfs.xfs /dev/zvol/tank/xfs
```

### Viewing RAID Status

```bash
zfs list
```

### Setting the Mount Point

```bash
zfs set mountpoint=/mnt/tank tank
```

### Data Recovery

```bash
zpool offline tank /dev/sdc
zpool status tank
zpool replace tank /dev/sdc /dev/nvme0n2
zpool status tank
```

**Explanation**: Takes a disk offline to simulate a drive failure, then replaces it with a new disk — RAID will automatically begin rebuilding.

### Decommissioning RAID

```bash
zpool export tank
```
