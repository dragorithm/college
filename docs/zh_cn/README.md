# 学习与作业共享仓库

## 仓库结构

### 配置文件

- **.cspell/technology.txt**: 开发过程中使用的技术词汇，用于 cspell 插件检查
- **.cspell/ensure.txt**: 确保文档中所有名称的拼写保持一致，包括人名、神名、龙族名称、宠物名称以及其他拼写固定的专有名词
- **.vscode**: VS Code（或 VSCodium）开发环境的配置文件

### 学院作业

- **college-linux**: 在学院完成的 Linux 相关作业
- **english**: 在学院完成的英语相关作业

### 学习过程笔记

- **zfs**: 学习 OpenZFS 时翻译整理的文档
- **git**：学习 Git 的笔记

## .gitignore

简单说明：`key` 目录中包含我的 GitHub SSH 密钥对。其他开发者不需要遵循这种做法。
